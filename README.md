# Journal of Applied Hermeneutics
repo for a TEI encoding and network analysis experiment

### TEI XML files from 2011-2025
These files have been converted from the open access PDF's available through the journal's website: https://journalhosting.ucalgary.ca/index.php/jah/issue/archive
The journal is published by the University of Calgary and was selected because of it's CC BY license which allows the CTL to use its materials without limiting restrictions.

The file formatting should follow the most current TEI guidelines from https://tei-c.org/release/doc/tei-p5-doc/en/html/index.html
